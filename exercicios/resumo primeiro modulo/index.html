3. Direito ao uso de imagens e seus formatos para web

É muito importante frisar que não temos direito de usar qualquer imagem no nosso site, e para isso devemos utilizar a ferramenta do google para achar imagens utilizáveis, ou usar sites como o Pexels.

Jpeg ou Png?

Bom, se você estiver procurando imagens que possam ser compactadas e ocupem pouco espaço ( o que facilita o carragemento de um site) utilize o formato JPG. Caso queira imagens que possam ter um fundo transparente, utilize o PNG

Tamanhos de imagem, GIMP e a tag img

Para desenvolver sites, tenha em mente que utilizar imagens muito grandes pode atrapalhar no SO do seu site, então sempre utilize imagens bem compactadas com larguras médias, como 600px, para não ter seu site PENALIZADO

Um aplicativo bom para mudar o tamanho das imagens é o GIMP, software gratuito para edição de imagens.

Como carregar imagens no seu código

Para carregar imagens no seu código, você pode salvá-las na pasta onde seu código está salvo e aperte Ctrl + Espaço, que sua imagem vai aparecer como uma opção, aí basta selecioná-la.

Outra opção é ir no google, copiar o endereço da imagem e colocá-lo no seu código
